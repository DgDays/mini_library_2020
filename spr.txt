/Главное окно

Это стартовое окно. В нём вы сможете добавить читателей,
либо изменить их характеристику.

Данное окно позволяет добавить, удалить, либо изменить
характеристику читателя

Кнопки, находящиеся сверху, позволяют сохранить файл с
учётом, получить доступ к окну с уведомлениями, изменить
тему, либо посмотреть информацию о сборке.

**Сочетания клавиш:
	Ctrl+A - добавить
	Ctrl+S - редактировать читателя
	Del - удалить читателя



/Окно Профиль

В это окне отображается вся информация о читателе и
книгах, которые он брал.
Также можно посмотреть статус книг, при этом строчка с
книгой будет окрашена в один из трёх цветов:
	Зелёный - книга сдана
	Красный - книга просрочена
	Жёлтый - книга на руках у читателя.

**В этом окне можно:
	1) Добавить книгу читателю
	2) Изменить статус книги ( на руках, просрочена, сдана )
	3) Удалить книгу

**Сочетания клавиш:
	Ctrl+A добавить книгу в ЧБ*
	Ctrl+S изменить книгу в ЧБ*
	Del - удалить книгу из ЧБ*

* ЧБ - читательский билет


/Окно Уведомления

В данном окне находится список книг, которые
просрочили читатели.



/Окно Учёта книг

В данном окне находится список всех книг, которые есть в
библиотеке. Они разделяются на учебники и художественную
литературу.

Учебники же делятся по предметам.

В этом окне есть возможность добавить книги, изменить
количество книг, либо удалить книги.

**Сочетания клавиш:
	Ctrl+A добавить книгу
	Ctrl+S редактировать значения книги
	Del - удалить книгу



/Окно Информации о книге

В этом окне можно посмотреть кто брал эту книгу и
вернул ли он ее в библиотеку.



/Окно Excel

Данное окно нужно для создания файла excel, в котором
будет сохранен учёт книг за выбранный вами промежуток времени.
